Race Into Space

Download this project as a .zip file Download this project as a tar.gz file

Race into Space is a free software version of the Liftoff! board game by Fritz Bronner, originally developed by Strategic Visions and published by Interplay in 1993 and 1994.

Here to play?

The most recent stable version of the game is available from the Race Into Space project on SourceForge. It runs natively on Windows, Mac OS X, and Linux; no DOSBox or other emulator is required.

Here to work?

A modernization of the 1.0 codebase lives in our GitHub repository. Hop in!
